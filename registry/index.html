How It Works

  1. We hash the original text using SHA-256 (the same algorithm Bitcoin uses)
  2. We anchor the hash to Bitcoin using OpenTimestamps - a free, open protocol
  3. We publish the proof file so anyone can verify independently
  4. The text is stored permanently (Arweave coming soon)

You don't have to trust us. You can verify the hash yourself, and confirm the Bitcoin timestamp using the OpenTimestamps verifier.
